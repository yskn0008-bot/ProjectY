# Taxi Operations v1 実用化リリース

期限：2026-07-18 17:00 JST

## 目的

沖縄夜勤タクシー営業を、iPhoneだけで営業前・営業中・営業終了まで一貫運用できる状態にする。

## 完成条件

- 営業開始、乗車、降車、休憩、メモ、営業終了をワンタップで記録できる
- 時刻・営業状態・乗車時間・休憩時間を自動記録できる
- 売上・乗車回数・平均単価・営業時給を自動計算できる
- 端末内へ自動保存し、オフラインでも使える
- YOSへ営業報告を共有できる
- ProjectY入力ログ互換CSVを出力できる
- ProjectY運行データとMission Controlへ移動できる
- GitHub Pagesで公開し、iPhoneホーム画面から起動できる
- 運転中の入力禁止をUIと運用ルールに明記する

## 構成

- YOS：最終判断、営業戦略、営業終了レビュー、ProjectYへの正式保存
- YOS Taxi Operations PWA：現場入力・状態管理・KPI・共有
- ProjectY運行データ：正本DB
- Mission Control：開発・問題・更新状況

## v1のデータ同期

PWAは認証情報を持たず、端末内へ安全に保存する。営業終了時にYOSへ共有し、YOSがProjectY運行データへ正式反映する。CSVバックアップも出力できる。

## 安全

操作は必ず停車中に行う。運転中は画面操作を行わない。
