# IMadaベース営業判断MVP

期限: 2026-07-18 17:00 JST

## 目的
YOS Taxiで、空車時間を中心に営業中の次の行動を短く判断できるようにする。

## 実装範囲
- 空車時間の自動計測
- 実車時間・実車率の自動計算
- 平均空車時間の自動計算
- 現在の空車時間に応じた「待機継続／移動準備／エリア変更」表示
- 残り目標と必要時給の表示
- 営業報告へKPIと判断内容を追加
- 既存の乗車・降車・休憩・GPS・CSVを維持

## 判断ルール
これはIMadaメソッドの考え方と、ようすけの既存運用ルールを組み合わせた当日運用MVP。

- 空車0〜5分: 現在地で反応観察
- 空車5〜10分: 同エリアで短時間検証
- 空車10〜15分: 反応が弱ければ移動準備
- 空車15分以上: エリア変更を優先

安全最優先。運転中操作は禁止。空車走行は最短にする。
