# YOS Mission Control Dashboard v1.0 運用ガイド

## 開く

GitHub Pages公開後、SafariでDashboardを開く。

## 基本操作

- 右上の`↻`：最新状態へ更新
- `概要`：注意項目と進行中を確認
- `全体`：全プロジェクトを確認・検索
- `Inbox`：承認待ち・確認待ちを確認
- `完了`：直近の完了を確認

## YOSで使う言葉

- `進捗`：DashboardとGitHubを確認して全体をまとめる
- `次`：今やることを一つに絞る
- `完了`：正本の状態と完了履歴を更新する
- `問題`：ブロッカーを登録する
- `保留`：理由と再開条件を記録する

## 正本更新

Dashboardは表示専用。変更はYOSが`data/mission-control.json`へ反映する。

## 通信できない時

前回保存した状態があれば表示する。赤い同期表示の場合、内容は最新ではない可能性がある。
