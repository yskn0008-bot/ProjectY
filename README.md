# YOS

YOS のメインプロジェクトです。iPhone での運用を主な前提として、プロダクトの方針・UI・素材・補助スクリプトを段階的に整理します。

> 現段階は初期セットアップです。本格的な機能実装や画面実装はまだ開始しません。

## 方針

- **既存資産を保護する**: 既存のファイルを削除・置換せず、改善を積み重ねます。
- **iPhone を優先する**: モバイル Safari での利用しやすさ、タッチ操作、縦向きの閲覧を重視します。
- **小さく進める**: 要件・導線・デザインを確認してから実装に入ります。

## ディレクトリ構成

```text
.
├── assets/     # 画像、アイコン、フォントなどの静的素材
├── docs/       # 要件、設計、運用に関するドキュメント
├── scripts/    # 開発・運用を補助するスクリプト
├── ui/         # UI の設計資料・画面資産・将来の実装
├── index.html  # エントリーポイント（保持）
└── README.md
```

各フォルダには用途を明確にしたファイルだけを追加してください。大規模な実装を始める際は、先に `docs/` に目的と判断を記録します。
